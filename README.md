# Info Project

This repository contains various files and a test agent setup for AI/ML experiments.

## Contents

- `git.md`: Git-related documentation
- `test.ts`: TypeScript test file
- `test agent/`: Directory containing AI/ML model configurations and scripts, including Hugging Face CLIP models

## Getting Started

Navigate to the `test agent/huggingface_openai_clip-vit-base-patch16_v5/` directory for model-specific instructions and requirements.

## Requirements

See `test agent/huggingface_openai_clip-vit-base-patch16_v5/requirements.txt` for Python dependencies.
